# PersistenceScript

Generate a persistence script for Metasploit embedded payloads.

## Installation

Use the Git to install PersistenceScript.

```bash
git clone https://github.com/rohitnishad613/PersistenceScript.git
```

```bash
-G    --GUI   : Start GUI version
```
